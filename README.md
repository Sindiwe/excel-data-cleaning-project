# Data Cleaning and Preparation in Microsoft Excel

## Project Overview

This project demonstrates the process of cleaning and preparing a messy dataset using Microsoft Excel. The dataset contained common data quality issues such as duplicate records, inconsistent formatting, extra spaces, blank cells, and incorrect values. The goal was to transform the raw data into a clean, reliable, and analysis-ready dataset.

## Objectives

* Improve data quality and consistency.
* Remove duplicate records.
* Correct formatting and data entry errors.
* Handle missing values and blank cells.
* Prepare data for analysis and reporting.

## Tools Used

* Microsoft Excel
* Excel Tables
* Find and Replace
* Sorting and Filtering
* Data Validation
* TRIM Function

## Data Cleaning Steps

### 1. Autofit Columns and Rows

Adjusted all columns and rows to display data clearly and improve readability.

### 2. Remove Duplicates

Identified and removed duplicate records using Excel's Remove Duplicates feature.

### 3. Trim Extra Spaces

Used the TRIM function to remove unnecessary spaces from text fields.

### 4. Handle Blank Cells

Located and addressed missing values by removing or updating blank cells where appropriate.

### 5. Convert Data into a Table

Converted the dataset into an Excel Table to improve organization, filtering, and data management.

### 6. Correct Errors

Used Find and Replace to fix spelling mistakes, inconsistencies, and formatting issues.

### 7. Validate Data

Reviewed the cleaned dataset to ensure accuracy, consistency, and completeness.

## Results

The final dataset is:

* Free of duplicate records
* Consistent in formatting
* Organized and structured
* Ready for analysis and reporting
* Improved in overall data quality
